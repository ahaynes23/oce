OCE

OpenCASCADE Community Edition (OCE): patches/improvements/experiments contributed by users over the official OpenCASCADE library.

About-History

The OCE project was started in march 2011, in order to set up a collaborative workspace (mailing list, git repository) able to gather and package contributions from the OpenCASCADE Technology (OCCT) community. OCCT is a well known open source development framework (C++) intended to 3D CAD, and provides industrial strength modeling algorithms, data exchange capabilities (STEP, IGES, STL) and many more powerful features. The OpenCASCADE company, editor of OCCT, recently announced (december 2011) and started a collaborative development portal, still under construction, in order to get closer to the community. The way to set up an efficient collaboration between OCE members (who are only a small part of the OCCT community) and this new service is currently being discussed.

OCE contributions mostly deal with portability issues: we take care to make installation easier (with the help of cmake) and to extend the list of supported platforms/compilers (OSX, latest linux/gcc versions, latest MSVC compiler, Borland compiler).

The OCE team releases a new version each month. The version number (starting from 0.1) is incremented (0.1, 0.2 ... 0.8) whatever the list of changes maybe. This release policy is not sync with OCCT releases: OCE needs some time to integrate OCCT changes, apply patches and package the release.
